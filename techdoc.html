Libraries

What are Libraries?

Libraries provide useful and efficient tools for programming.

List of C++ Libraries:

  • iostream
  • fstream
  • vector
  • string
  • STL

How to Import a Library:

#include "iostream"

The library iostream allows the code to print onto and take inputs from the console.

Static Typing

What is Static Typing?

In C++, a data type declaration is required to instantiate a variable.

Examples of Static Typing:

int x = 10;
double d = 10.0;
string s = "Hello World";
char c = 'p';

Benefits:

Static Typing allows the compiler to catch compilation errors at compilation time instead of runtime. In short, error finding is faster.

If Statements

What are if statements?

If statements serve to impose a condition before running a piece of code. C++ if statements are also syntactically similar to Java and Python.

if (credit_score > 700) { borrow(); }

The statement above translates: if your credit score is above 700, you can borrow money using the borrow function.

Benefits:

If statements are very useful for checking if something is true.

Loops

What are loops?

Loops continously repeat a segment of code. C++ loops are also syntactically similar to Java and Python.

Type of loops

  • For Loop

    • For loops works by iterating from a beginning to an end value.

    for(int index = 0; index < total; index++) { doSomething(); }

    In this example, the iterator (the value that gets incremented from every loop iteration) begins at 0 and increments by 1 continously until it reaches the end. For each incrementation, the loop performs whatever is defined between the brackets.

  • While Loop

    • While loops will continuously run whatever is defined in the brackets as long as its conditional statement in parenthesis is true.

    while(merchandise_count < 100) {merchandise_count++;}

    In the example above, the while loop stockpiles the merchandise by 1 as long as the number of merchandises is less than 100.

  • Ranged For Loop

    • Ranged for loops works by looping a segment a code as long as it can still read from an object.

    for(int price: receipt) { total_price += price;}

    In the example above, as long as the loop is reading a price from the receipt, it will add the price to the total price. When it reaches the end of the receipt, the loop will terminate.

    Function

    Dealing with large batches of code can be overwhelming. Functions can help organize and simplify your code.

    A function is like a mathematical function where it takes in inputs and outputs something. It is important to define the return type of a function before defining the function (e.g void, int, double, etc.). Then, you will need to provide a function name that makes sense (what the function is doing). Every function should be able to take in parameters, by definition of functions. So, after the function name, the () operator follows and should take in parameters. Finally, the code that the function runs should follow and be encapsulated within brackets.

    int main() {
    int merchandise_count = 0;
    while (merchandise_count < 100) { merchandise_count++;}
    cout << "merchandise count" << merchandise_count;
    }
    int fill_merchandise(int num_merchandise){
    while (num_merchandise < 100) {num_merchandise++;}
    return num_merchandise;
    }
    int main() {
    int merchandise_count = 0;
    cout << "merchandise count" << fill_merchandise(merchandise_count);
    }

    In main(), the number of lines of code has decreased. This makes our code readable because we pushed away how the stockpiling works to a function.
